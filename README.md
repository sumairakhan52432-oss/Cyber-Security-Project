# Cyber Security Project

## 📌 Project Title
Cyber Security Lab Project using Kali Linux, Wireshark and AhMyth

## 📖 Description
This project is based on practical cyber security concepts and lab work. It includes theoretical explanations and practical implementation of different cyber security techniques such as footprinting, reconnaissance, malware analysis, and network scanning.

The project report is uploaded as a PDF file which contains complete explanation, commands, and screenshots of the practical work.

## 🛠 Tools Used
- Kali Linux
- Wireshark
- Nmap
- AhMyth RAT
- Android Virtual Device (Emulator)

## 🔍 Topics Covered
- Footprinting and Reconnaissance
- Active vs Passive Information Gathering
- Malware and its types (Virus, Worm, Trojan, Ransomware)
- Network Scanning using Nmap
- Packet Analysis using Wireshark
- Remote Access using AhMyth (Android RAT)

## 🧪 Practical Work
The project includes hands-on lab tasks:
- Nmap scanning of target system
- Capturing network packets in Wireshark
- Creating Android APK using AhMyth
- Connecting Android device with Kali Linux server

All practical steps and screenshots are included in the PDF report.
